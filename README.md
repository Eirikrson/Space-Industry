# Space Industry

Space Industry is a 2D browser-based factory and exploration game set in space.

Build and expand your spaceship, mine asteroids, explore star systems, manage resources and research new technologies. Create automated production chains and grow your rocket fleet across the galaxy.

## Features

* Ship building system
* Resource mining
* Factory automation
* Research system
* Space exploration
* Combat and defense systems
* Procedurally generated star systems

## Play Online

https://eirikrson.github.io/Space-Industry/

## Development Status

Space Industry is currently in active development. Features and gameplay may change between versions.
